# Facet Studio v3

Generador de patrones de facetas para joyería y objetos 3D plegados.

**[→ Abrir la app](https://mvalls12.github.io/facet-studio/)**

## Funciones
- 8 tipos de objeto (plano, cono, cilindro, cuenco, cúpula, jarrón, anillo, perfil libre)
- 18 siluetas de joya
- 8 tipos de patrón (estándar, chevron, ondas, radial, ladrillo, rombos…)
- Simulación 3D con animación de pliegue
- Exportar PNG 300dpi, SVG vectorial
- Calculadora de material (área, perímetro, merma)
- Diseños guardados en sesión

## Uso
Abre directamente en el navegador. No requiere instalación ni servidor.
